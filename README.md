# Literature Logs

Welcome to my personal knowledge base. I created this repository to log my learnings, literature reviews, and foundational notes throughout my PhD journey.

Because research is an iterative process of reading, testing, and refining, I use this space as a living document to track the concepts and methodologies I encounter. My work primarily intersects Intelligent Transportation Systems, Reinforcement Learning, Generative AI, and software verification. Instead of just bookmarking papers, this is where I synthesize relevant metrics, summarize literature, and break down complex architectures for my own reference.
